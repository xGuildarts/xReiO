أدخل بيانات Supabase لربط النظام بالداتابيز

2. Settings → APIانسخ الـ Project URL والـ anon key
3. SQL Editorشغّل ملف supabase_setup.sql
4. للتحديث الجديد (فوجين)شغّل: ALTER TABLE mtr_bookings ADD COLUMN IF NOT EXISTS convoy2 integer; ALTER TABLE mtr_bookings ADD COLUMN IF NOT EXISTS seat_assign2 jsonb DEFAULT '[]';
5. للتحديث الجديد (جهة التحصيل)شغّل: ALTER TABLE mtr_bookings ADD COLUMN IF NOT EXISTS collect_via text;
